복잡한 수집을 한 문장으로

노드를 일일이 잇지 않아도 됩니다. 무엇을 모을지 말하면 RPA·컴퓨터유즈·파일검색을 엮어 연결된 플로우를 만들어 줍니다.

01

화면을 보는 봇

로그인이 필요한 사이트도 한 번 로그인하면 세션을 기억합니다. 선택자가 없으면 컴퓨터유즈가 화면을 보고 직접 클릭합니다.

02

중복은 알아서

식별 컬럼(제목·URL)을 키로 두면 이미 가져온 항목은 다시 쌓이지 않습니다. 수치가 바뀌어도 한 건만 유지합니다.

03

스키마도 AI로

담을 정보를 말하거나 CSV·엑셀을 올리면 탭별로 컬럼·타입·키를 자동으로 잡아 줍니다.

04

내 데이터, 내 컴퓨터

모든 건 로컬 SQLite에 저장되고, 토큰으로 보호된 data API로만 외부에 제공됩니다. 도메인 노출은 터널로 한 번에.

05

표는 진짜 DB처럼

컬럼 추가·제약·정렬·필터·CSV 내보내기. 행을 직접 편집하거나 봇·커넥터로 채웁니다.

06

커넥터 마켓플레이스

Naver·Coupang·Meta·Amazon·Apify 등 키 기반 커넥터를 설치하듯 연결하고 채널별 수집 주기를 지정합니다.

설치하고 바로 시작

앱 하나면 됩니다. 자동화 엔진은 첫 실행 때 알아서 준비됩니다.

1

다운로드 & 열기

받은 파일을 응용 프로그램으로 옮기고 실행합니다.

2

첫 실행 준비

자동화에 쓰는 브라우저를 한 번 내려받습니다. 이후엔 바로 봇 실행.

3

모으기 시작

채팅으로 봇을 만들거나 커넥터를 연결하면 로컬 DB에 데이터가 쌓입니다.

macOS에서 “손상되었다”거나 “확인되지 않은 개발자” 안내가 뜨면(서명 전 초기 빌드라 격리된 것뿐입니다), /Applications로 옮긴 뒤 터미널에서 한 번 실행하세요:
xattr -dr com.apple.quarantine "/Applications/Nora Intake.app"
